## Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/premiumstoic/basil.git
   cd basil
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Setup environment variables**
   
   Create a `.env` file in the root directory:
   ```env
   VITE_NEON_DATABASE_URL=your_neon_database_url_here
   JWT_SECRET=your_random_secret_key_here
   ```

4. **Setup database**
   
   Run the SQL in `schema/users.sql` and `schema/cards.sql` in your Neon database console.

5. **Run development server**
   ```bash
   npm run dev
   ```

## Deployment

See [VERCEL_DEPLOYMENT.md](./VERCEL_DEPLOYMENT.md) for complete deployment instructions.

**Quick deployment:**
1. Push to GitHub
2. Import repository on [Vercel](https://vercel.com/new)
3. Add environment variables (`VITE_NEON_DATABASE_URL`, `JWT_SECRET`)
4. Enable Vercel Blob storage
5. Deploy!

## Environment Variables

| Variable | Description | Required |
|----------|-------------|----------|
| `VITE_NEON_DATABASE_URL` | Neon PostgreSQL connection string | Yes |
| `JWT_SECRET` | Secret key for JWT tokens | Yes |
| `BLOB_READ_WRITE_TOKEN` | Auto-generated by Vercel | Yes* |

*Auto-generated when you enable Vercel Blob storage
